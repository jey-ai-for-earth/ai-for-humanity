# AI for Humanity - AI for Earth

Local prototype website for education about careful AI use and its environmental and human impact.

## Open locally

Open `index.html` in a browser, or run `run.bat` to start a local server.

## Publish

This repo is set up for GitHub Pages deployment from the `main` branch.
After the workflow runs, the site should appear at:

`https://jey-ai-for-earth.github.io/ai-for-humanity/`

## What is included

- A strong landing page with a clear education message
- Sections on why AI use has a cost
- Role-based guidance for individuals, teams, and educators
- Scenario examples and a checklist
- A small interactive pledge section
- A Node-based local server helper for testing
- GitHub Pages deployment workflow
